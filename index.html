Data Explorer

Welcome to the EOX Data Explorer

This app allows you to visually inspect your orders. Click on the button to show a list of the available datasets you can explore.

The dataset is displayed on the map, which you can zoom and pan as you like. Using the button you can adjust the visualization by selecting the bands to use and processing steps to apply.

For the band selection, either define red, green, and blue channels individually or use one of the predefined scenarios.

This app uses the geotiff.js library to read the actual raster values like UInt16 from the band files (or the overviews, depending on the maps zoomlevel). Taking advantage of the optimized structure of COGs, only the parts of the file that are actually required are requested, drastically reducing the overall amount of data transmitted from the archive.

If you want to know more about COGs please refer to their introduction page.
